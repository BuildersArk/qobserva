# qobserva-local

**One-command local stack for QObserva:** starts the collector and dashboard so you can run and view quantum run events locally.

> ⚠️ **Beta:** This package is in beta. APIs may change.

## Install

```bash
pip install qobserva-local
```

## Quick start

```bash
qobserva-local up
```

This starts the collector and UI; the dashboard URL is printed in the terminal. To stop:

```bash
qobserva-local down
```

Use with `qobserva-agent` to instrument your quantum code and send events to this stack.
